Project-Glu

Project-Glu is my attempt glue some existing things together to make managing my son's T1D a little easier/safer. Currently only focused on data from the G4, but later I'd like to include pump/activity tracker data.

current status

I have a forked version of @jcostik's original-android-app running on my old/rooted Galaxy Nexus a new Moto G and have it posting simple json to this node.js/express.js/angular.js/mongo app.

I the android app posting to the web app running locally and at AWS.

I also have a forked version of NightScout running at AWS with the same mongo db as the basic web-app.
